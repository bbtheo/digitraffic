# digitraffic: R Client for the Finntraffic Digitraffic TMS (LAM) API

Provides functions to access traffic measurement data from the Finnish
Transport Infrastructure Agency's Digitraffic open API. Covers TMS
(Traffic Measurement System, known as LAM in Finnish) stations including
real-time sensor data, station metadata, sensor calibration constants,
and historical raw vehicle-level observations. All data is returned as
tidy tibbles ready for analysis.
