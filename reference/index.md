# Package index

## Station Metadata

Discover and inspect LAM measurement stations

- [`dt_stations()`](https://bbtheo.github.io/digitraffic/reference/dt_stations.md)
  : List all LAM/TMS measurement stations
- [`dt_station()`](https://bbtheo.github.io/digitraffic/reference/dt_station.md)
  : Get detailed metadata for a single LAM/TMS station
- [`dt_stations_load_details()`](https://bbtheo.github.io/digitraffic/reference/dt_stations_load_details.md)
  : Fetch and cache detailed metadata for all LAM/TMS stations

## Sensor Data

Real-time traffic measurements

- [`dt_station_data()`](https://bbtheo.github.io/digitraffic/reference/dt_station_data.md)
  : Get real-time sensor data for a single LAM/TMS station
- [`dt_stations_data()`](https://bbtheo.github.io/digitraffic/reference/dt_stations_data.md)
  : Get real-time sensor data for all LAM/TMS stations
- [`dt_sensors()`](https://bbtheo.github.io/digitraffic/reference/dt_sensors.md)
  : List all available LAM/TMS sensor types

## Sensor Constants

Station calibration and reference values

- [`dt_sensor_constants()`](https://bbtheo.github.io/digitraffic/reference/dt_sensor_constants.md)
  : Get sensor calibration constants for LAM/TMS stations

## Historical Data

Raw historical vehicle-level observations

- [`dt_history_raw()`](https://bbtheo.github.io/digitraffic/reference/dt_history_raw.md)
  : Fetch raw historical vehicle-level data for a LAM/TMS station

## Utilities

Helper functions

- [`dt_vehicle_classes()`](https://bbtheo.github.io/digitraffic/reference/dt_vehicle_classes.md)
  : Vehicle classification codes used in LAM raw data
- [`dt_cache_clear()`](https://bbtheo.github.io/digitraffic/reference/dt_cache_clear.md)
  : Clear the in-session digitraffic cache
